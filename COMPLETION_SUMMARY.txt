================================================================================
USAGE INSTRUCTIONS
================================================================================

1. INSTALLATION
   cd Codebase_Sprint1
   pip install -r requirements.txt

2. GENERATE SAMPLE DATASET
   python generate_dataset.py
   (Creates 20 synthetic images with annotations)

3. TRAINING
   python main.py --model simple --epochs 50 --batch_size 32
   python main.py --model resnet --learning_rate 0.0005
   python main.py --model yolov3 --epochs 100

4. INFERENCE
   python inference.py --model checkpoints/best_model.pth --image path/to/image.jpg
   python inference.py --model checkpoints/best_model.pth --image_dir path/to/images/

5. CONFIGURATION
   Edit config.yaml to customize:
   - Dataset paths
   - Model parameters
   - Training hyperparameters
   - Augmentation settings
